📡 Project 4 – REST API with Express

This project focuses on creating a fully functional REST API using Node.js and Express. It provides endpoints to serve structured JSON data, handle route parameters, query strings, and return meaningful error messages.

🔧 Technologies Used

  • Node.js
  • Express
  • JavaScript Modules (ES6)

📁 Key Files

  • p4-server.js – Sets up the server and routes
  • p4-module.js – Handles request logic and responses
  • p4-data.js – Static dataset returned by the API
  • package.json – Project metadata and dependencies

🚀 How to Run

  1. Install dependencies: npm install
  2. Run the server: npm start
  3. Open browser or API tester and go to http://localhost:3000/api/data or similar routes
